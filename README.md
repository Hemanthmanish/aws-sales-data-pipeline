AWS Sales Data Pipeline

This project demonstrates an end-to-end data engineering pipeline built on AWS.

Technologies:
AWS S3
AWS Glue
PySpark
Amazon Redshift
SQL

Architecture:
Raw Data → S3 → Glue ETL → Redshift → Analytics

Features:
• Automated ETL pipelines using AWS Glue
• Data transformation using PySpark
• Data warehouse design using Amazon Redshift
• Analytics queries for sales insights
